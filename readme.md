# 📈 CurrWeather: Интерактивная крипто-аналитика

Веб-приложение для отслеживания курсов валют с возможностью закрепления виджетов и отправки сводных отчетов на email.

## 📂 Структура проекта
* `/Front`, `/CSS`, `*.html` — Клиентская часть (Vanilla JS, HTML5, CSS3)
* `/Back` — Серверная часть (Python, Flask, smtplib)

## 🚀 Инструкция по локальному запуску

### 1. Подготовка и запуск Backend-сервера
Убедитесь, что у вас установлен Python 3.8+. Откройте терминал в корневой папке проекта.

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
Перейдите в папку бэкенда:

Bash
cd Back
Создайте файл .env на основе .env.example и внесите настройки SMTP-сервера (Яндекс/Mail.ru).

Запустите сервер:

Bash
python email_service.py
Сервер будет доступен по адресу http://127.0.0.1:5000

2. Запуск Frontend-клиента
После того как терминал сообщит об успешном запуске Flask-сервера, просто откройте файл htmlpage.html (находящийся в корневой папке) в любом современном браузере.
